# View File History

See all revisions of a specific file.

## File History View

Open any versioned file and the **File History** pane in the Source Control sidebar updates automatically to show its revision history.

Each revision shows:

- Revision number
- Author
- Date
- Commit message

## Actions

Right-click a revision to:

- **Open diff** — compare with previous revision
- **Open this revision** — view that exact version
- **Copy revision number**
- **Copy message**
- **Rollback to this revision**

## Tip

Use file history to understand why code changed over time.
