# View File History

See all revisions of a specific file.

## Open File History

**Option 1: Context Menu**

1. Right-click a file in Explorer
2. Select **SVN: Show Log**

**Option 2: Command Palette**

1. Open the file
2. Run **SVN: Show commit messages**

## File History View

Shows for each revision:

- Revision number
- Author
- Date
- Commit message

## Actions

Click a revision to:

- View changes (diff)
- Open that version
- Copy revision number
- Copy commit message

## Tip

Use file history to understand why code changed over time.
