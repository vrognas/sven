# Select Folders to Download

Use the Selective Download view to manage which folders are local.

## Browse Server Contents

1. Open **Selective Download** view in Source Control
2. Expand folders to see their contents
3. Items are fetched from server on-demand

## Visual Indicators

- **Normal text**: Downloaded locally
- **De-emphasized**: On server only (not downloaded)
- **Checkmark**: Included in sparse checkout

## Include/Exclude Folders

**To download a folder:**

- Click the folder
- Select **Include** or **Checkout**

**To exclude a folder:**

- Right-click the folder
- Select **Exclude**

## Refresh

Click **Refresh** to sync the view with the server.

## Tip

Start with minimal folders, add more as needed.
