# Compare Revisions

View differences between file versions.

## From File History

1. Open File History view
2. Click any revision
3. See diff against previous revision

## Compare Options

Right-click a revision for more options:

- **Open Diff**: Compare with previous
- **Open Diff with BASE**: Compare with your local base
- **Open File (Remote)**: View that exact version

## Reading Diffs

- **Red lines**: Removed
- **Green lines**: Added
- **Blue highlights**: Changed within line

## External Diff Tool

Configure an external diff tool:

```json
"svn.diff.useDiffTool": "external",
"svn.diff.externalTool": "code --diff"
```

## Tip

Compare revisions to understand the evolution of code.
