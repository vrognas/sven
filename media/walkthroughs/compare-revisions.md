# Compare Revisions

View differences between file versions.

## From File History

1. Open File History view
2. Right-click any revision
3. Select **Open diff** to compare with previous revision

## Compare Options

Right-click a revision for more options:

- **Open diff**: Compare with previous
- **Open diff with BASE (Your Last Update)**: Compare with your local base
- **Open this revision**: View that exact version

## Reading Diffs

- **Red lines**: Removed
- **Green lines**: Added
- **Blue highlights**: Changed within line

## Tip

Compare revisions to understand the evolution of code.
