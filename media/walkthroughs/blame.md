# Blame Annotations

See who last modified each line of code.

## Toggle Blame

1. Open a file
2. Run **SVN: Toggle Blame** from Command Palette
3. Or use keyboard shortcut (if configured)

## Reading Annotations

Each line shows:

- Revision number
- Author name
- Relative date

## Hover for Details

Hover over any annotation to see:

- Full commit message
- Exact date and time
- Link to view the full commit

## Customize Appearance

Settings to adjust blame display:

- `svn.blame.dateFormat`: Date format
- `svn.blame.layout.showAuthorName`: Show/hide author
- `svn.blame.layout.showDate`: Show/hide date

## Tip

Blame helps find the right person to ask about code changes.
