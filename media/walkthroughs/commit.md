# Commit Changes

Save your changes to the repository.

## Steps

1. Stage your changes (or commit all)
2. Enter a commit message in the input box
3. Press `Ctrl+Enter` / `Cmd+Enter` to commit
4. Or click the **âœ“** checkmark icon

## Good Commit Messages

- Start with a verb: "Add", "Fix", "Update", "Remove"
- Be specific: "Fix login timeout on slow networks"
- Keep the first line under 72 characters

## Commit Options

- **Commit Staged**: Only staged files
- **Commit All**: All modified files
- **Pick Commit Message**: Reuse previous messages

## After Committing

Your changes are now in the repository. Others can see them after they update.
