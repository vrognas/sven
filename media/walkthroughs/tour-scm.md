# Source Control Panel Overview

The SVN extension adds 4 views to the Source Control panel.

## Views

| View                   | Purpose                       |
| ---------------------- | ----------------------------- |
| **Repo History**       | Full commit log               |
| **File History**       | Revisions for selected file   |
| **Branch Changes**     | Commits unique to your branch |
| **Selective Download** | Sparse checkout management    |

**Remote Changes** appear as a section in the main SCM panel when server updates are available.

## Quick Access

- Click the **Source Control** icon in the Activity Bar (left side)
- Or press `Ctrl+Shift+G` / `Cmd+Shift+G`

## Tip

Collapse views you don't use often by clicking their headers.
